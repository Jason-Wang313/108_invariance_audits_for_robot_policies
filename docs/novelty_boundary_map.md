# Novelty Boundary Map

## Crowded Territory

- Generic augmentation.
- Generic equivariant policies.
- Generic domain randomization.
- Generic invariant risk minimization.
- Generic uncertainty or conformal shift filtering.
- Benchmark-only contribution.

## Claimed Boundary

Invariance audits for robot policies classify transformations by physical validity, preserving valid perceptual/geometric invariances while rejecting transformations that change contact frame, force direction, material/friction, tool affordance, support surface, or temporal order.

## What Would Falsify The Claim

If augmentation, equivariance, domain randomization, invariant risk minimization, or conformal shift filtering matches the proposed audit on success, invalid-invariance rejection, false-invariance acceptance, contact violation, unsafe action, and ablation robustness under external robot validation, the paper should be killed.
