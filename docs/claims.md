# Claims

## Supported Local Claim

Robot-policy invariance should be audited for physical action equivalence. Observation-level invariance can be harmful when a transformation changes contact frame, affordance, support surface, material response, temporal order, or safety preconditions.

## Frozen v5 Evidence

The v5 audit beats the strongest non-oracle baseline, `proposed_physical_invariance_audit_v4`, on hard success, utility, invalid-invariance rejection, false acceptance, contact-frame violation, unsafe action, paired seeds, ablations, stress sweep, and strict fixed-risk utility.

## Unsupportable Claim
