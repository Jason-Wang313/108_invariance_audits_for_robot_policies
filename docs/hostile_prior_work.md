# Hostile Prior Work

The hostile set contains equivariant perception, domain randomization, invariant risk minimization, contrastive representation learning, transporter-style robot manipulation, and conformal shift filtering.

The novelty boundary is narrow. The contribution cannot be "use invariance", "use equivariance", "augment data", "domain randomize", or "detect distribution shift." The surviving claim is physical-validity auditing: deciding which invariances preserve the action and which erase contact frame, force direction, material, affordance, or temporal-order distinctions.

The v4 local benchmark supports this boundary but does not close the external-validation gap.
